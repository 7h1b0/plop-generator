# Plop generator

Plop Generator is an image generator web app.

- :art: Change primary color
- :eye: Press `Space` to move the eye

# Developing

To develop:

1. Clone the repository
1. To install node packages, run:
   ```sh
   npm install
   ```
1. Start the development server by running:
   ```sh
   npm run dev
   ```
